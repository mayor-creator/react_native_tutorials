# TipCalculator

A simple and elegant tip calculator application built with React Native and Expo. This app helps you easily calculate tips and split bills among friends.

## Features

-   **Bill Entry**: Enter the total bill amount.
-   **Tip Selection**: Choose from preset tip percentages (5%, 10%, 15%, 25%, 50%) or enter a custom tip amount.
-   **Split Bill**: Specify the number of people to split the bill with.
-   **Real-time Calculation**: Instantly see the tip amount per person and the total amount per person.
-   **Reset**: Quickly reset all fields to start a new calculation.

## Tech Stack

-   **React Native**: For building the mobile user interface.
-   **Expo**: For easy development and deployment.
-   **TypeScript**: For type safety and better developer experience.

## Getting Started

### Prerequisites

-   Node.js installed on your machine.
-   Expo Go app installed on your iOS or Android device (or an emulator).

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd TipCalculator
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the App

Start the development server:

```bash
npx expo start
```

-   **iOS**: Press `i` in the terminal or scan the QR code with the Camera app (if using Expo Go).
-   **Android**: Press `a` in the terminal or scan the QR code with the Expo Go app.
